# Документация по ролям системы

Эта папка содержит подробную документацию для каждой роли в системе СпортБаш.

## Список ролей

1. **[Спортсмен (Athlete)](ROLE_ATHLETE.md)** - Роль для пользователей, занимающихся спортом
2. **[Тренер (Coach)](ROLE_COACH.md)** - Роль для тренеров спортивных организаций
3. **[Родитель (Parent)](ROLE_PARENT.md)** - Роль для родителей, отслеживающих спортивную деятельность детей
4. **[Директор организации (Director)](ROLE_DIRECTOR.md)** - Роль для директоров спортивных организаций
5. **[Сотрудник спорткомитета (Committee Staff)](ROLE_COMMITTEE_STAFF.md)** - Роль для сотрудников городского комитета по физической культуре и спорту
6. **[Главный администратор (Admin RB)](ROLE_ADMIN_RB.md)** - Роль для главного администратора системы

## Структура документации

Каждый файл документации содержит:

- **Описание роли** - общее описание назначения роли
- **Получение роли** - как пользователь может получить эту роль
- **Данные для входа** - информация для входа в систему с этой ролью
- **Функционал** - подробное описание всех возможностей роли
- **API Endpoints** - список API endpoints, доступных для роли
- **Ограничения** - что роль не может делать
- **Особенности** - уникальные особенности роли

## Общая информация

### Мультиролевая система

Система поддерживает **мультиролевую модель**, где один пользователь может иметь несколько ролей одновременно. Например:
- Спортсмен + Родитель
- Тренер + Спортсмен
- Директор + Тренер

### Переключение ролей

Пользователи с несколькими ролями могут переключаться между ними в личном кабинете через переключатель ролей.

### Общие данные

Все роли одного пользователя используют общие данные:
- ФИО
- Фото профиля
- Email
- Телефон
- Город

Изменение этих данных отражается во всех ролях.

### Уникальный ID роли

Каждая роль имеет уникальный 8-символьный ID (буквы и цифры), который используется для:
- Связи родителя с ребёнком
- Идентификации роли
- Поиска пользователей

## Получение ролей

### Свободно доступные роли:
- **Спортсмен** - выбирается при регистрации
- **Тренер** - выбирается при регистрации

### Автоматически создаваемые роли:
- **Родитель** - создаётся автоматически при добавлении ребёнка
- **Директор** - создаётся автоматически при одобрении организации

### Роли по коду/одобрению:
- **Сотрудник спорткомитета** - регистрация по специальному коду
- **Главный администратор** - назначается администратором системы
