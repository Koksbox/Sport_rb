## Как использовать

### 1. Загрузить тестовые данные
```bash
python manage.py load_sample_data --clear
```

### 2. Запустить сервер
```bash
python manage.py runserver 0.0.0.0:8060
```

### 3. Открыть сайт
```
http://127.0.0.1:8060
```

### 4. Попробовать анонимный просмотр
- Откройте `/organizations/` - список организаций доступен без входа
- Откройте `/events/` - список мероприятий доступен без входа

### 5. Зарегистрироваться
- Откройте `/register/`
- Заполните форму
- **Выберите тип кабинета** (обязательно!)
- После регистрации автоматически откроется соответствующий кабинет

## Типы кабинетов

1. **Спортсмен** - для тех, кто хочет заниматься спортом
2. **Родитель** - для родителей детей, занимающихся спортом
3. **Тренер** - для тренеров
4. **Организация** - для представителей спортивных организаций
